# Null-Line Hypothesis Framework

## Core Thesis
The null line (k . k = 0) in Minkowski space is the single primitive from which all geometric, physical, and mathematical structure is generated.

## Key Claims
1. **Primitive Trinity**: Triangle (3 lines @ 120 degrees), Square (4 lines @ 90 degrees), Circle (limit of n lines). These map to ADE Lie algebra classification.
2. **Observer**: The observer is a null line (photon absorption event). Matter = bound null lines.
3. **Riemann Zeta**: zeta(s) is a null-field partition function. The critical line sigma=1/2 is the midpoint of the two-sided null line (k+, k-).
4. **Hilbert-Polya**: A self-adjoint operator H_null on Twistor Space (PT+) yields eigenvalues matching zeta zeros.
5. **String Theory**: H_null spectrum reproduces bosonic string mass spectrum (M^2 ~ N-1). Calabi-Yau 3-fold unifies the three primitives.

## Critical Assessment
- **Strengths**: Synthesizes twistor theory, string amplitudes (Witten 2003), and number theory. Elegant analogy between null two-sidedness and functional equation zeta(s)=zeta(1-s).
- **Weaknesses**: 
  - **Contradiction**: Body claims RH proof; Coda admits 'We have not proved the Riemann Hypothesis. The Eisenstein wall stands.'
  - **Geometry**: Mapping Euclidean tilings to Minkowski null lines lacks standard geometric derivation.
  - **Spectral**: Linking H_null eigenvalues to string mass spectrum requires non-trivial justification (dimensions, quantization).

## Research Directions

- Simulate null-line interference patterns to test 'matter' generation.
- Verify Galois group 2T connection to Euler product.
